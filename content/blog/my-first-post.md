这是我的第一篇博客。其实想写博客这件事已经很久了，但一直被各种理由拖延——直到上周，我终于下定决心，把这件事落地了。

## 为什么是现在

想法和行动之间，差的往往只是一个契机。上周末整理书签时，发现自己收藏了几百篇技术文章。我突然意识到，与其一直做信息的消费者，不如也开始做一个信息的创造者。

于是我对自己说：别等了，就现在。

## 技术选型

搭建个人网站，技术选择有很多选项：

- **静态站点生成器** — Hugo、Jekyll、Astro 等，功能强大
- **CMS 系统** — WordPress、Ghost 等，开箱即用
- **纯静态 HTML** — 手写，最简单也最灵活

最终我选择了纯静态 HTML + CSS + JavaScript。原因有三：

1. 个人博客的内容量不大，纯静态完全够用
2. 不需要依赖任何框架或构建工具，维护成本最低
3. 可以完全掌控每一行代码，是一种很好的学习方式

## 设计思路

在设计上，我遵循了"内容优先"的原则。博客的核心是文字，所以设计应该为阅读服务，而不是喧宾夺主。

具体来说：
- 内容区域宽度限制在 720px，保证舒适的阅读体验
- 使用系统字体，减少加载时间
- 蓝灰配色，干净整洁，支持暗色模式
- 响应式设计，移动端也能良好阅读

## 踩过的坑

虽然是一个简单的静态网站，但在搭建过程中还是遇到了一些小问题：

- **相对路径** — 文章详情页在 blog 目录下，引用 CSS 和 JS 时需要用 ../ 回到上级目录
- **暗色模式闪烁** — 如果 JS 加载较慢，会先显示亮色再切换到暗色，需优化加载顺序
- **移动端导航** — 固定顶部导航在手机上的交互需要仔细设计

> "Any sufficiently advanced technology is indistinguishable from magic." — Arthur C. Clarke

## 下一步计划

博客的初始版本已经上线，但还有很多可以改进的地方：

- 增加更多文章内容
- 优化网站性能
- 添加更多交互效果
- 考虑接入更高级的评论系统
